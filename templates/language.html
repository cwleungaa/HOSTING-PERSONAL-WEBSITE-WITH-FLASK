Language

This section introduces how an AI can be constructed to process human language.

Context-Free Grammar
In Context-Free Grammar, the text is abstracted from its meaning to represent the structure of the sentence using formal grammar.
In python, we can use library nltk to implement this idea(Project: Parser).

Information Retrieval
Information retrieval is the task of finding relevant documents in response to a user query.
We use tf-idf, by considering term frequency and Inverse Document Frequency, to assign a value to each word.
This method can output a list of words per document that are likely to span the main topics that define the document(Project: Questions).
