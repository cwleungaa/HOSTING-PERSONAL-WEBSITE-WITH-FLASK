Knowledge

This section introduces how AI can base on exiting knowledge to draw conclusion using Propositional Logic and inference.

Propositional logic is based on propositions, statements about the world that can be either true or false.

Inference is the process of deriving new sentences from old ones.

Project:

1. Knights
Write a program to solve logic puzzles.
EG: A says “I am both a knight and a knave.”
The AI has to deduce that A is a liar since A can’t be both knight and knave at the same time.
