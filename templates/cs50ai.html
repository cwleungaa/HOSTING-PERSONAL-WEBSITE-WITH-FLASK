CS50’s Introduction to Artificial Intelligence with Python

This course introduces the concepts and algorithms of modern artificial intelligence.
This course mainly focuses on the following topics:

Search
It’s about finding a solution of problems like finding the best path between two places or finding the best move of a game.

Knowledge
It’s about how AI base on existing knowledge to draw conclusion using Propositional Logic.

Uncertainty
It’s about how AI makes optimal decisions using Probability.

Optimization
It’s about choosing the best option from a set of possible option.

Learning
It’s about Machine Learning - how computer learns to recognize patterns and able to execute tasks on its own.

Neural Networks
A program structure inspired by the human brain that is able to perform tasks effectively.

Language
It’s about how AI process human language.

Each topic is followed by one or two projects in order to practice the theory of each topic.
