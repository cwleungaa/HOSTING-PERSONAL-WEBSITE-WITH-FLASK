Final Project (RSA Message Encryption and Decryption System)

In this final project, I am using python and SQL to build an RSA message encryption and decryption system.
This program allows users to send secret messages between sender and receiver.

This program contains four files, two python files is the encryption and decryption program.
One csv file contains the encrypted messages.
And one SQL file contains users account information, including username, password, public key and secret key.
The user interface is built by using the tkinter modulus of python.

The encryption method is called RSA algorithm, which I learnt in my Math Number theory class.
The method mainly is assigning a pair of unique public key and secret key to each user.
To encrypt the message, the algorithm will use sender’s secret key and receiver’s public key to do the encryption.
After that, for decryption, we need to know the sender of message, in order to pick the correct sender’s public key, with his own receiver’s secret key to do the decryption.
This algorithm prevent from sending message to incorrect person because they can’t decrypt the message using their own receiver secret key.
Also, the receiver has to know the identity of the sender in order to pick the correct sender’s public key.
