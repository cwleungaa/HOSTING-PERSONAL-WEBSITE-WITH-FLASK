Search

This session introduces different kind of search method.
It includes:

Uninformed search algorithms that do not utilize any knowledge about the problem that they did not acquire through their own exploration.
EG: Depth-First Search, Breadth-First Search.

Informed search algorithm considers additional knowledge to try to improve its performance.
EG: Greedy Best-First Search, A* Search

Adversarial Search which the algorithm faces an opponent that tries to achieve the opposite goal.
EG:Minimax algorithm, Alpha-Beta Pruning
