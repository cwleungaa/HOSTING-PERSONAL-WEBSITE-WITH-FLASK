Optimization

Optimization is choosing the best option from a set of possible options.
This section introduces various tools and algorithm on handling optimization problems.

Linear Programming:
Linear programming is a family of problems that optimize a linear equation (an equation of the form y = ax₁ + bx₂ + …).

Using the scipy library in Python, with all the variables inserted into the program properly, it can find the optimal solution if the solution exists.

Constraint Satisfaction
Constraint Satisfaction problems are a class of problems where variables need to be assigned values while satisfying some conditions.
With the understanding of Node consistency and Arc consistency, it introduces the Backtracking search algorithm.
